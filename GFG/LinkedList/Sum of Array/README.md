## Problem Statement

Given an integer array `arr[]`, return the sum of all elements in the array.

## Examples

**Input:**

```text
arr = [1, 2, 3, 4]
```

**Output:**

```text
10
```

**Input:**

```text
arr = [1, 3, 3]
```

**Output:**

```text
7
```

## Approach

Initialize a variable `sum` to `0`. Traverse the array and add each element to `sum`. After processing all elements, return the final value of `sum`.

## Algorithm

1. Initialize `sum = 0`.
2. Traverse the array.
3. Add each element to `sum`.
4. Return `sum`.

## Time Complexity

**O(n)**
